# GlyphSearch

Search for icons from Font Awesome, Foundation, Glyphicons, IcoMoon, Ionicons, Material Design, and Octicons at [glyphsearch.com](http://glyphsearch.com).

## Build

```
npm run build:all
```

to build linux, win, mac app in `optput` folder
